# Osama Café ☕
### Premium Artisanal Coffee Roastery & Café Landing Page

Welcome to the **Osama Café** project repository! This project is **Task 2** developed as part of the **NTI Full Stack Developer using PHP** training program. 

It is a modern, premium, and fully responsive landing page designed to showcase an artisanal coffee roastery, built from scratch using clean **HTML5** and **Vanilla CSS3** with a focus on rich aesthetics, visual excellence, and smooth user experience.

## ✨ Features

- **Cozy & Premium Aesthetics:** Designed with a curated dark-themed coffee aesthetic, blending deep warm tones, smooth gradients, and elegant typography (Outfit/Inter).
- **Fully Responsive Layout:** Optimized for all screen sizes (Desktops, Laptops, Tablets, and Mobile phones) using CSS Flexbox and CSS Grid.
- **Interactive Preloader:** A smooth, custom page loading screen featuring the café logo to elevate the initial user impression.
- **Modern Hamburger Navigation Menu:** A responsive, checkbox-based CSS hamburger menu for mobile devices without requiring heavy JS libraries.
- **Rich Sections:**
  - **Hero Section:** Captivating visual intro with smooth call-to-actions (Explore Menu, Visit Us).
  - **Our Story:** Details about bean sourcing (100% organic, ethical) with beautiful icon lists.
  - **The Roastery:** Side-by-side section highlighting the roasting process with vintage-inspired styling.
  - **Specialties Menu:** Grid layout showcasing specialties (Espresso, Slow Brew, Pastries) with hover animation cards.
  - **Contact & Map:** Integrated Google Map iframe alongside contact details (Address, Phone, Email).
  - **Comprehensive Footer:** Organized widget columns for values, hours, contact info, and developer credit.
- **SEO Best Practices:** Semantic HTML5 structure (header, nav, section, footer), correct heading hierarchy, metadata, keywords, and accessibility attributes.

## 🛠️ Technologies Used

* **HTML5:** Semantic markup structure.
* **Vanilla CSS3:** Custom properties (CSS variables) for theme consistency, custom grid systems, and transition-based micro-animations.
* **Font Awesome v6:** Premium vector icons for menu links and details.
* **Google Fonts:** Integration of modern, high-quality typography.
* **Google Maps API:** Embed responsive iframe map location.

## 🚀 How to Run the Project Locally

No compiler or framework installer is required. You can run this project directly in your web browser:

1. **Clone or Download the Repository:**
   ```bash
   git clone https://github.com/Osama2214/NTI-Task-2.git
   ```
2. **Open the project folder.**
3. **Double-click `index.html`** to open it in your default web browser (Chrome, Edge, Firefox, Safari).
4. *Alternatively (Recommended):* If using VS Code, right-click `index.html` and select **"Open with Live Server"** to enable auto-refresh on changes.
